## Sistema de Cadastro de Produtos em Python

Este exercício apresenta um sistema simples de cadastro de produtos, onde o usuário informa o nome e o preço, e o programa armazena tudo em um dicionário.
É um exercício ideal para treinar:

- Estruturas de repetição (while)
- Tratamento de erros com try/except
- Armazenamento de dados em dicionários
- Entrada e validação de dados
- Conversão de tipos (float)
- Controle de fluxo com continue e break

---

## Descrição do Exercício

- O programa permite cadastrar produtos de forma repetida.
- A cada ciclo o usuário deve informar:
    Nome do produto
    Preço do produto (valor numérico)
- Após cada cadastro:
    Os dados são adicionados ao dicionário produtos.
    Todos os itens cadastrados são exibidos.
    O usuário decide se deseja continuar cadastrando ou encerrar o programa.
- O código também trata erros, como:
    Nome de produto vazio
    Preço digitado incorretamente (texto, vírgulas, caracteres inválidos)

---

## Como Executar o Programa

Salve o código em um arquivo, por exemplo:

```cadastro_produtos.py```

Execute no terminal:

```python cadastro_produtos.py```

Digite o nome e preço dos produtos conforme solicitado.
