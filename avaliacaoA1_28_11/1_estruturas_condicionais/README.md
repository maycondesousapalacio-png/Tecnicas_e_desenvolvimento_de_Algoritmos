## Verificação de Idade para Entrada em Eventos

Este projeto contém um simples programa em Python que simula um sistema de verificação de idade para permitir (ou negar) a entrada de um participante em um evento.
O objetivo é treinar o uso de entrada de dados, tratamento de erros, casting de tipos e condicionais.

---

## Descrição do Exercício

O programa solicita a idade de um participante e verifica:

- Se o valor digitado é válido (apenas números).
- Se a idade é menor que 18 anos, negando a entrada.
- Se a idade é igual ou maior que 18 anos, permitindo a entrada.

Também demonstra o uso de:

- input() para coleta de dados
- isalpha() para prevenir erros
- Conversão de string para inteiro
- Estruturas condicionais if, elif
- Mensagens de retorno ao usuário

---

## Como Executar o Programa

Certifique-se de ter o Python 3 instalado.

Salve o código em um arquivo, por exemplo:

```verificar_idade.py```


Execute o programa no terminal/cmd com:

```python verificar_idade.py```

Digite a idade quando solicitado.
