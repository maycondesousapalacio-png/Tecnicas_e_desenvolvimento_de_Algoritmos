## Contador de Números Pares com for e while

Este exercício demonstra duas formas de criar contadores em Python que exibem apenas os números pares de 1 a 100.
Ele utiliza:

- Estrutura de repetição for
- Estrutura de repetição while
- Operador módulo (%) para identificar números pares
- Controle de fluxo e incremento

É um exercício ideal para praticar loops e condições simples.

---

## Descrição do Exercício

O programa deve:

- Utilizar um laço for para percorrer os números de 1 a 100 e imprimir apenas os pares.
- Utilizar um laço while para obter o mesmo resultado.
- Ambos os métodos demonstram formas diferentes de resolver o mesmo problema.

---

## Como Executar o Programa

Salve o código em um arquivo, por exemplo:

```contador_pares.py```

Execute no terminal com:

```python contador_pares.py```

Você verá os números pares de 2 a 100 sendo impressos.
