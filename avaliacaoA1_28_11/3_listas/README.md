## Cadastro de Alunos em Lista

Este exercício apresenta um programa simples em Python que coleta nomes de alunos, armazena em uma lista e, ao final, exibe todos eles organizados em ordem alfabética.
Ele treina:

- Uso de listas
- Estruturas de repetição (while)
- Funções de validação (any, isdigit)
- Ordenação de listas (sort())
- Exibição formatada com enumerate

---

## Descrição do Exercício

O programa permite cadastrar nomes de alunos repetidamente.
A cada ciclo:

- O usuário informa um nome.
- O programa valida:
    Se o nome contém números (não permitido).
    Se o campo está vazio.
- O nome é adicionado à lista de alunos.
- O usuário escolhe se deseja continuar cadastrando.
- Ao final:
    A lista é ordenada em ordem alfabética.
    Todos os alunos são exibidos com numeração.

---

## Como Executar o Programa

Salve o código como:

```cadastro_alunos.py```

Execute no terminal:

```python cadastro_alunos.py```

Digite os nomes conforme solicitado.
